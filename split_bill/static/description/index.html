Split Bill Module

Advanced bill splitting with complex dynamics

Key Features

  • Multiple Split Methods: Split bills equally, by percentage, by item, by amount, or with custom rules
  • Delivery Fee Handling: Distribute delivery fees equally, proportionally, or assign to a specific payer
  • Discount Allocation: Apply discounts equally, proportionally, or to specific items
  • Complex Splitting: Support for complex splitting scenarios with multiple participants

Detailed Features

  • Split Sessions: Create multiple split sessions for a single bill to try different splitting approaches
  • Participant Management: Add, remove, and manage participants with detailed tracking
  • Item Allocation: Assign items to specific participants or split them among multiple participants
  • Payment Tracking: Track payments from each participant with multiple payment methods

Integration

  • Sales Integration: Import items directly from sales orders
  • Invoice Integration: Import items from invoices
  • POS Integration: Import items from POS orders
  • API Access: Programmatic access to split bill functionality
